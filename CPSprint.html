About the Event

Reverse Coding

In this round, participants will be provided with an executable file created from a pre-determined logical code. Their task is to reverse-engineer the logic and write the code themselves to replicate the functionality. This challenge will test participants' analytical and coding skills under time constraints.

Start: 11 Jan 25, 09:00 PM IST
End: 11 Jan 25, 10:00 PM IST

Competitive Programming

Focused on data structures and algorithms, this round will challenge participants to solve complex problem statements across multiple difficulty levels. Participants will need to write efficient and optimized code to address a series of progressively difficult tasks, testing their problem-solving abilities under time pressure.

Start: 11 Jan 25, 10:00 PM IST
End: 11 Jan 25, 11:30 PM IST

Final Round

The Final Round is a 4-way competitive programming battle between the top four teams, featuring 3 intense questions relating to CP and DSA.

Start: 12 Jan 25, 09:00 PM IST
End: 12 Jan 25, 11:30 PM IST
